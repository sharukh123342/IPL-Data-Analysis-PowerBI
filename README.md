# IPL Data Analysis Dashboard (2008-2024)

## Project Overview

This project analyzes IPL matches from 2008 to 2024 using Power BI.

## Tools Used

- Power BI
- SQL
- Excel
- Power Query
- DAX

## Dashboard Pages

### Page 1 - Tournament Overview
- Total Matches
- Total Seasons
- Match Type Distribution
- Top Cities
- Top Venues

### Page 2 - Team Performance
- Most Successful Teams
- Toss Analysis
- Win Type Analysis

### Page 3 - Batting & Bowling Analysis
- Total Fours
- Total Sixes
- Runs Per Over
- Wickets Per Over
- Extras Analysis

## Key Insights

- Mumbai Indians are among the most successful teams.
- League matches dominate IPL history.
- Batting and bowling performance varies significantly across teams.
